## 🔍 找图技巧

### 推荐源（按清晰度排序）
1. **出版社官方页面** (Springer Link / Routledge / Wiley) — 编辑过的最高清
2. **Google Books** (https://books.google.com) — 有 zoom=5 的原图
3. **京东 / 当当 / 淘宝** 商品页 — 国内电商常有商家上传的高清图
4. **亚马逊** (https://www.amazon.com) — 出版社上传的 print-quality 图
5. **Open Library** (https://openlibrary.org) — 公共版权图

### 京东/当当/淘宝检索关键词模板
```
书名 + ISBN
例: "Regulatory Aspects of Gene Therapy and Cell Therapy Products 978-3-031-34567-8"
```

### 命名规范（重要！）
发图时**直接用上面"建议保存名"作为文件名**，我会自动匹配到 INDUSTRY_BOOKS 数组中对应条目。

### 文件要求
- 格式：JPG / PNG 都可以
- 尺寸：≥ 800px 宽即可（建议 1500px+）
- 大小：≥ 200KB 优先（确保清晰度）
- 命名：按"建议保存名"保存
