## 手动打包方法

### Windows 资源管理器
1. 打开 `C:\Users\YSJ\WorkBuddy\Claw\`
2. 进入 `stemcell-kb` 文件夹
3. 全选 `index.html` 和 `assets` 文件夹
4. 右键 → 发送到 → 压缩(zipped)文件夹

### PowerShell 命令
```powershell
Compress-Archive -Path "C:\Users\YSJ\WorkBuddy\Claw\stemcell-kb\index.html","C:\Users\YSJ\WorkBuddy\Claw\stemcell-kb\assets" -DestinationPath "C:\Users\YSJ\WorkBuddy\Claw\stemcell-kb_website.zip" -CompressionLevel Optimal
```

### 7-Zip
```bash
7z a stemcell-kb_website.zip index.html assets
```

## 本地预览方法

打包后解压到任意目录：
- **方法 1**: 双击 `index.html`（直接浏览器打开，部分功能依赖 file:// 可能受限）
- **方法 2**: 用本地服务器（推荐）

```bash
# Python 3 自带服务器
cd 解压目录
python -m http.server 8080

# 然后浏览器打开 http://localhost:8080
```
